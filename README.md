# SBER_Hack_task_variant_generator

# Микросервис analyze

## Зависимости
    requirements.txt

    Для установки: pip install -r requirements.txt

## .env
    Путь к базе данных со справочными таблицами для классификации, например:
    DATABASE_URL=sqlite:///data/tasks.db

## структура database/

    app/database/
    │
    ├── db.py - Отвечает за подключение к БД и создание сессий.
    ├── models.py - Модели (таблицы) базы данных
    ├── init_db.py - Создание таблиц.
    └── seed_db.py - Заполнение базы начальными данными через ORM

## Создать базу данных со справочными таблицами
    Перейти в папку микросервиса analyze и запустить в командной строке
    python -m app.database.init_db

## Заполнить справочную базу данных
    Перейти в папку микросервиса analyze  и запустить в командной строке
    python -m app.database.seed_db

## Запустить сервис
    Перейти в папку микросервиса analyze  и запустить в командной строке
    uvicorn app.main:app --reload

    запустится микросвервис на http://127.0.0.1:8000
    можно перейти на http://127.0.0.1:8000/docs и передать POST запрос вида: 
    {
        "text": "Решите уравнение 2x + 5 = 15"
    }
    и посмотреть ответ
